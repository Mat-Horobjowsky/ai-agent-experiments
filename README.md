# AI Text File Agent

A lightweight Python agent that scans text files, extracts structured insights, and outputs JSON results.

## What it does
- Scans a folder for `.txt` files
- Extracts:
  - word count
  - short preview
  - file size
- Outputs structured JSON

## Why this matters
This pattern is reusable for:
- document analysis
- compliance review
- data preprocessing
- automation pipelines
